This project is still in active development, so don't expect your game to be fully compatible just yet. However, games that use no PulpScript (i.e. only use the default dialog options for sprites and items), should already be working!

Support for sound and music are not yet implemented, and inputs like the crank and accelerometer will probably be out of scope for the first release.
