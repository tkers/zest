# Zest Apps

## development utilities

Apart from the core runtime at the heart of this project, here are some handy extras:

- **[Bundler](/zest/bundler)**\
   Easy-to-use tool to wrap your game in a standard template and export as single HTML file.

- **[Cartography](/zest/maps)**\
   Tool to automatically generate a world map from your game, or export all rooms individually.

- **[CLI (NodeJS)](/zest/cli)**\
   Command line interface for several Zest features.

---

- **[Embed](/zest/embed)**\
  Example showing an embeddable Zest game.

- **[Jukebox](/zest/juke)**\
   Example showing a simple music player for Pulp songs.
